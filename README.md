# Product Sales Analysis Dashboard
I built an interactive dashboard in Excel by taking the sales data from an FMCG MNC linked below. The dashboard shows top KPIs such as Total revenue figures by country, city and region. It also shows Total Orders, Average Order Values, Median revenue for products, and Gross Margins.  The dashboard is created in the form of a dropdown list where the key metrics displayed change in real time based on the options selected. This was done using Data Validation to create dropdown.   This project taught me to clean data for analysis and to calculate and report the top KPIs. I used several Excel formulas such as Sum, Filter, Unique, Sort, AverageIF, SumIFs, IF, etc. The theme and design of the dashboard was created editing chart elements, and I chose colours that matched and did not contrast so as to minimise distraction.
